# 📱 WolfPlay Apps

Repositório oficial dos aplicativos **WolfPlay**.

Aqui você encontra os APKs disponíveis para download manual no Android.

## 📦 Downloads

Acesse a página de releases para escolher o aplicativo que deseja baixar:

[⬇️ Ver APKs disponíveis](https://github.com/canaiswolfplay/wolfplayapps/releases)

## 🔄 Atualizações

Cada aplicativo pode ter sua própria release.

Para atualizar um APK mantendo o mesmo link, edite a release correspondente, remova o arquivo antigo e envie o novo APK com o mesmo nome.

## ⚠️ Aviso

Para instalar um APK no Android, talvez seja necessário permitir a instalação de apps de fontes desconhecidas nas configurações do aparelho.

Baixe sempre pelos links oficiais deste repositório.
